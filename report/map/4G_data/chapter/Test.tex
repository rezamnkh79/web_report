\section{Call Test Results}
he calls test can be of two types:
\begin{itemize}
\item 
Short calls should last the average of a user call - a good reference value is 15 to 30 seconds. Serve to check whether the calls are being established and successfully completed (being a good way to also check the network setup time).
\item 
Long calls serve to verify if the handovers (continuity between the cells) of the network are working, ie calls must not drop.
\end{itemize}
